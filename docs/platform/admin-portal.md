# Admin Portal

The **Admin Portal** is the central command center for your Enterprise Express Agent environment. It provides a unified interface for enterprise administrators to manage all aspects of their organization's AI implementation.

![Admin Portal Dashboard](../assets/images/eea_admin_portal.png)

## Key Capabilities

From the Admin Portal, you can:
- **Manage Account Settings**: Update enterprise details, subscription plans, and global configurations.
- **Workspace Management**: Create, configure, and delete workspaces to organize agents by department or function.
- **User & Group Administration**: Invite users, assign roles (RBAC), and organize them into groups for easier policy management.
- **Permissions**: Define granular access controls to ensure security and compliance.
- **Agent Management**: View a global list of agents, monitor their health, and manage their lifecycles.
- **Status & Observability**: Access real-time dashboards showing agent status, usage metrics, and system health.
