# Create an Agent

Agents are the core of the EEA platform. Here is how to create a basic agent.

## Configuration Steps

1. Navigate to the **Agents** tab in your workspace.
2. Click **New Agent**.
3. Fill in the following details:
    - **Name**: `SupportHelper`
    - **Description**: `Assists with common customer support queries.`
    - **Model**: Select `GPT-4o` (or preferred model).
    - **Instructions**: `You are a helpful support assistant. Answer queries concisely.`
    - **Authentication**: Select `None` (for public/anonymous agents).
4. Click **Create Agent**.
